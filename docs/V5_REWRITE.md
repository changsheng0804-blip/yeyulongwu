# V5 重写记录

## 重写原因
原 V5 将形状作为"背景叠加"——只选取 2400 个粒子形成形状,其余粒子仍在背景中流动。用户体验:形状只是背景上的一个点缀,不是过场效果。

## 重构变化

### 核心理念转变
全部 N 个粒子参与形状过场:公式 → 聚合(0.85s) → 定形(2.4s,带微小呼吸感) → 溶散(0.55s) → 回公式

### 状态机
```
idle → appear → hold → dissolve → idle
```

### 与音乐同步
- 乐章切换时(s===0 && bar===0 && pos>0)自动触发随机形状过场
- 每种音乐风格有偏好的形状列表(风格导向):
  - 中国风: 心形/星/圆脸/炸裂
  - 布鲁斯: 心形/舞步/双螺旋/无穷
  - 电子: 漩涡/双螺旋/炸裂/星
  - 爵士: 无穷/心形/圆脸/星

### 保持期"呼吸感"
hold 期间 blend 不锁死在 1.0,而是 0.95 + sin(...)*0.03 的微弱震荡,形状看起来在"呼吸"而非死板静止

### 手动触发
8 个形状按钮仍可用(手动点触发不过场逻辑,纯手动触发)
