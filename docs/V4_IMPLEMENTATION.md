# V4 实现总结

## 已实现

### A. 骨架音 + 填充音
- 手写总谱保留不动，写好的每个音以完整音量播放
- 休止位 (`-`) 由 V2 级进生成器以 55% 音量填充
- 填充概率 = 0.08 + (16-骨架音数)×0.05 × (0.5+0.5×粒子动能)
- `melodyDeg` 向下一骨架音方向级进 (`Math.sign(d)`)
- 若无下一个骨架音（如第四小节末尾），小幅随机漂移

### B. 参数正弦漫游
- 每乐章定义 base + wander + rate 三元组
- spring/swirl/trail 在基准值附近正弦游走，不偏离 MC 安全区间
- 不同相位+不同倍率的正弦波混合，避免机械重复

### C. 贝斯极低频粒子微振
- 贝斯发声时 bassEnv=1，衰减常数 3/s
- 渲染循环中：`vz[i] += sin(t*18 + i*0.01) * bassEnv * 0.03`
- 用于暗涌×环面绸带预设的"潜知觉振动"设计

### D. 乐章间图案融化过渡
- 切换乐章时 transBlend=0, transSteps=32
- target() 函数内：混合 oldPreset 和 preset 两套公式的目标坐标
- 32 帧内线性过渡完成
- 同时参数在过渡期内采用加权混合，避免突变

## 新增：画面状态指示

左下角双行显示：
1. 当前乐章标题
2. "⟳ 图案融化中 N%" 或 "粒子动能 0.xx · 参数漫游中"
