# V5 重写 v2 — 文字成形 + 自动复位视角

## 改动

### 1. 移除低辨识度形状
- 移除: 😊 大圆脸、🕺 机械舞步
- 保留: 💚 心形、⭐ 五角星、🧬 双螺旋、∞ 无穷、🌀 漩涡、💥 炸裂

### 2. 新增: 粒子文字成形
- 离屏 Canvas(512×160)渲染用户输入的中/英文字体
- 自适应字号: `measureText` → 调整 fontSize 使文字占满 88% 宽度
- 白色像素采样 → 等比缩放映射到 3D 空间(XY 平面写字, Z 轴 0.5 单位厚度)
- Fisher-Yates 洗牌消除采样顺序痕迹
- 输入框默认值"鱼龙",支持 Enter 键触发
- 代码中 shapeIdx=99 表示文字成形

### 3. 自动复位最佳视角
- 每种形状预设最佳观察角度(SHAPE_BEST_ANGLE[])
  - 心形/星/无穷/漩涡/文字: yaw=0(正面), pitch≈0.15
  - 双螺旋: yaw=0.4(略侧面才能看清双链)
- 触发形状时: 记录当前视角 → 自动 lerp 到最佳角度
- 定形结束后溶散时: 自动 lerp 回自由视角
- 用户手动拖动鼠标时: 立即放弃自动复位(camAlive=false)
- 自动旋转在复位期间暂停

### 4. 文字绑定到音乐过场
- 中国风/电子/布鲁斯的乐章自动触发列表中加入了 99(文字)
- 每轮循环可能随机抽到"鱼龙"文字过场
