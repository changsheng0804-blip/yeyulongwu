# 一夜鱼龙舞 · Night of Dancing Lights

> 基于参数方程实时生成的粒子光流动画，支持三维光雕渲染与图形状态驱动的自生成音乐。

## 版本

| 版本 | 在线预览 | 说明 |
|------|---------|------|
| **V2 三维光雕** | [🌐 在线预览](https://changsheng0804-blip.github.io/yeyulongwu/v2/) | 3D 透视渲染 + 四套立体图案 + 音画互驱引擎 |
| V1 意境版 | [🌐 在线预览](https://changsheng0804-blip.github.io/yeyulongwu/) | 2D Canvas 参数曲线粒子光流 |

## V2 特性

- 🎨 四套三维图案预设：鱼龙舞 / 球谐光雕 / 环面绸带 / 星涡
- 🎵 图形状态驱动的自生成音乐（五声音阶旋律 + 和弦进行）
- 🔁 音画闭环：粒子运动 → 音乐生成，节拍脉动 → 视觉呼吸
- 🖱️ 拖拽旋转视角 · 滚轮缩放 · 点击触发冲击波 + 锣声
- ⚡ 帧率无关的弹簧力物理模拟 + Float32Array 内存优化
- 🎛️ 7 个实时可调参数（形态/速度/密度/跟随力/漩涡力/光迹/音量）
- 📱 响应式，桌面/移动端自适应
- 🚫 零外部依赖，单文件 HTML

## 部署

直接推送到 GitHub 仓库，在 Settings → Pages 中启用 GitHub Pages，即可获得在线链接。

V2 位于 `v2/` 子目录，需要分别启用：
- 分支 `main`，路径 `/ (root)` → V1 页面
- V2 作为根目录下的子路径，同一个 Pages 部署生效
