# 一夜鱼龙舞 · Night of Dancing Lights

> 「鱼龙舞于夜，一夜成形」—— 一个极具沉浸感的 Web 粒子音乐可视化实验项目。数千粒子在三维空间中舞动，随着音乐自动聚合成文字与几何形状，配合四种不同风格的程序合成音乐，带来独特的视听体验。

## V5 特性

- 🔤 **粒子文字成形**：输入中文实时生成立体粒子文字，高精度采样 + 智能字体兜底，字形边界清晰锐利，支持最多 6 字
- 🎭 **粒子成形过场**：乐章切换时 8000 粒子自动聚合→定形→溶散，四阶段状态机驱动
- 📷 **自动复位最佳视角**：成形时自动转至正面最佳观察角度，溶散后恢复自由视角
- 🎵 **四种音乐风格**：🇨🇳 中国风羽调 / 🎸 布鲁斯蓝调 / 🔊 电子合成波 / 🎷 爵士慢摇，每种含四乐章完整总谱
- ✨ **六种几何形状**：心形、五角星、双螺旋、无穷符号、漩涡、炸裂
- 🖱️ **实时交互**：拖拽旋转、滚轮缩放、点击冲击波
- 🎯 **视觉细节**：定形阶段消除拖尾重影、鼓三件套差异化视觉反馈、状态栏实时提示
- 🚫 **纯前端实现**：零外部依赖，单文件 HTML

## V3 特性 (Fable 5 · 乐章版)

- 🥁 四轨总谱：底鼓/军鼓/踩镲 + 贝斯走根音 + 和弦垫 + 主旋律
- 🎼 字符式记谱法：一个字符 = 一个十六分音符，直接可编辑
- 🎯 音画在音频时钟 (`AC.currentTime`) 上精确对齐，非帧计数
- 🖼️ 三种鼓的差异化视觉角色：底鼓膨胀+镜头震，军鼓抖动+色移，踩镲微光
- 🎭 四乐章结构：星涡序曲 → 鱼龙行板 → 环面急板 → 球谐辉煌
- 🔄 乐章间图案/物理参数/音色同步切换，第四乐章开场自带锣声

## V2 特性

- 🎨 四套三维图案预设 + 蒙特卡洛搜索的临界态优化预设 (Top 3)
- 🎵 图形状态驱动自生成音乐（五声音阶旋律）
- 🔁 音画闭环：粒子运动 → 音乐生成，节拍脉动 → 视觉呼吸
- 🖱️ 拖拽旋转视角 · 滚轮缩放 · 点击触发冲击波 + 锣声
- ⚡ Float32Array 内存优化 + 帧率无关物理模拟
- 📱 响应式，桌面/移动端自适应
